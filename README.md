## पण्डित गोविन्द झा | मैथिल ब्राह्मण पुरोहित

यह परियोजना एक साधारण **स्टेटिक (static) सिंगल‑पेज वेबसाइट** है, जो पण्डित गोविन्द झा जी की वैदिक एवं धार्मिक सेवाओं, ऑनलाइन सेवा, कैलेंडर, प्रशंसापत्र और बुकिंग/संपर्क विवरण को प्रदर्शित करती है। पूरा UI मुख्यतः हिन्दी/देवनागरी में है और किसी सर्वर‑साइड कोड या डेटाबेस पर निर्भर नहीं है।

### टेक स्टैक

- **HTML5**: पूरी साइट `index.html` में लिखी गई है।
- **CSS3 (inline `<style>` के रूप में)**: रंग योजना, लेआउट, टाइपोग्राफी और एनिमेशन के लिए।
- **Vanilla JavaScript**: नेविगेशन, सेक्शन‑स्क्रॉलिंग, फॉर्म हैंडलिंग और स्क्रॉल‑बेस्ड ऐनिमेशन के लिए।
- **Google Fonts**: देवनागरी और हेडलाइन के लिए कस्टम फ़ॉन्ट्स।

### प्रोजेक्ट संरचना

- **`index.html`**  
  - फिक्स्ड नेवबार और मोबाइल‑फ्रेंडली हैमबर्गर मेनू  
  - हीरो सेक्शन (नाम, परिचय, श्लोक आदि)  
  - About, ऑनलाइन सेवाएँ, पारंपरिक सेवाएँ, पंचांग/कैलेंडर, प्रशंसापत्र, FAQ  
  - बुकिंग/कॉन्टैक्ट फॉर्म (नाम, तिथि, स्थान इत्यादि)  
  - सरल स्क्रोल‑ऐनिमेशन (Intersection Observer के माध्यम से)  
  - बुकिंग फॉर्म के लिए JS हैंडलर, जो क्लाइंट‑साइड पर ही सफलता संदेश दिखाता और फॉर्म रीसेट करता है

इसके अलावा किसी अन्य `src/`, `backend/` या कॉन्फ़िगरेशन डायरेक्टरी का प्रयोग नहीं किया गया है।

### कैसे चलाएँ (Run)

यह एक साधारण स्टेटिक पेज है, इसलिए कोई इंस्टॉलेशन आवश्यक नहीं है।

- **सीधा ब्राउज़र में खोलें**  
  - रिपोज़िटरी क्लोन करें या ZIP डाउनलोड कर के अनज़िप करें।  
  - `index.html` पर डबल‑क्लिक करें या ब्राउज़र (Chrome/Firefox/Edge आदि) में `Open File` से खोलें।

- **लोकल HTTP सर्वर (अनुशंसित)**  
  कुछ ब्राउज़र `file://` के माध्यम से लोड की गई फ़ाइलों पर सीमाएँ रखते हैं, इसलिए लोकल सर्वर बेहतर अनुभव देता है।

  - Python 3:
    ```bash
    cd pandit_govind_jha
    python -m http.server 8000
    ```
    फिर ब्राउज़र में `http://localhost:8000/index.html` खोलें।

  - Node.js (यदि इंस्टॉल हो):
    ```bash
    cd pandit_govind_jha
    npx serve .
    # या
    npx http-server .
    ```
